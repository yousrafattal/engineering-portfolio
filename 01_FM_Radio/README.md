# FM Radio Receiver

This project was the first that actually made me question, wait what is inside an IC? Which then led to the endless "smaller universes" problem of electronics. It's like PCBs are already a world of their own and now there's is also a whole universe inside an  IC.

Finished radio
└── PCB
    ├── resistors, capacitors, buttons
    └── radio IC package
        └── silicon die
            └── circuit blocks
                └── transistors
                    └── semiconductor regions and atomic structure
