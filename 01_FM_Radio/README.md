# FM Radio Receiver

1. Project Overview

A battery-powered, digitally tuned FM stereo receiver kit built around an RDA5807FP receiver IC. The radio uses push-button controls for power, volume, and station seeking, and the connected headphone cable also functions as the FM antenna.

This project was the first that actually made me question, wait what is inside an IC? Which then led to the endless "smaller universes" problem of electronics. It's like PCBs are already a world of their own and now I've opened my eyes to the universe of millions of transistors that make up semiconductors.
```text
Finished radio
 └─ PCB
  └─ resistors, capacitors, buttons
   └─ radio IC package
     └─ silicon die
        └─ circuit blocks
           └─ transistors
              └─ semiconductor regions and atomic structure
```
